# Security Policy

## Supported Versions

Only the latest version of ElectroBot AI is supported for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it via the GitHub Issues page or contact the maintainer directly. 

We take security seriously and will investigate all reports.

### What to report:
- Potential leaks of API keys.
- Content Security Policy (CSP) bypasses.
- Rate limiting failures.
- Improper input sanitization.

Thank you for helping keep ElectroBot AI secure for all citizens!
