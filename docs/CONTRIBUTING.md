# Contributing to ElectroBot AI

Thank you for your interest in contributing to ElectroBot AI! We welcome contributions that help improve the education of Indian citizens about the electoral process.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally.
3. **Create a branch** for your feature or bug fix.
4. **Make your changes** and ensure code quality.
5. **Add tests** for any new functionality.
6. **Submit a Pull Request** with a clear description of your changes.

## Code Standards
- Follow JSDoc standards for all functions.
- Ensure all sensitive keys are kept in `.env` and never committed.
- Maintain the Glassmorphism design aesthetic.

We look forward to your contributions!
